El color comunica intención. Para una identidad creíble: contraste AA/AAA, jerarquía clara y acentos consistentes.

  • Base neutra para legibilidad (#111827 sobre #F9FAFB).
  • Acento principal violeta #6D28D9 + acento secundario verde #10B981.
  • Usa el acento para CTAs y highlights, no para todo.
